# Security Policy

## Supported Versions

The table below indicates the versions of this project that currently receive security updates.

| Version | Supported |
|---------|:---------:|
| Latest Release | ✅ Yes |
| Older Versions | ❌ No |

---

## Reporting a Security Vulnerability

The security of this project is important. If you discover a security vulnerability, please report it responsibly.

Please **do not disclose security issues publicly** until they have been reviewed.

When reporting a vulnerability, include the following information:

- A clear description of the issue.
- Steps to reproduce the vulnerability.
- The potential impact.
- Any relevant screenshots or logs (if applicable).
- Suggestions for possible fixes (optional).

---

## Response Process

After receiving a security report, the following process will be followed:

1. Acknowledge the report.
2. Review and verify the issue.
3. Develop and test a fix if required.
4. Release an updated version of the project.
5. Credit the reporter where appropriate (unless anonymity is requested).

---

## Scope

This security policy applies to:

- Arduino source code
- Project documentation
- Repository configuration

Third-party libraries used by this project maintain their own security policies.

---

## Responsible Disclosure

Please help keep this project secure by reporting vulnerabilities responsibly and avoiding public disclosure until the issue has been resolved.

Thank you for helping improve the security and reliability of this project.
